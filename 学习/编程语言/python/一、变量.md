## 一、变量类型

### 1、简介

*python中，可以使用  “ =” 进行赋值操作*

*赋值过后的计算就会涉及到变量的类型，变量类型不同，就不能直接运算*

变量类型分为str、int、list、tuple和dict

```python
# 变量类型
name = 'john'
age  = 25
list = [5,10,15,20]
tuple = (5,10,15,20)
dict = {'name':'john','age':'25'}

>>>type(name)
<class 'str'>

>>>type(age)
<class 'int'>

>>>type(list)
<class 'list'>

>>>type(tuple)
<class 'tuple'>

>>>type(dict)
<class 'dict'>
```
